# G2 Innovation - Proyecto de Ingeniería de Software

Aplicación web desarrollada para la materia Ingeniería de Software.

## Ejecución del proyecto

1. Instala las dependencias:
	```
	npm install
	```
2. Inicia la aplicación:
	```
	npm start
	```
